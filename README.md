# ngx-sse-client

A simple **SSE** (Server Sent Events) client for `Angular` applications. More information an how to use [here](lib/projects/ngx-sse-client/README.md) and [here](https://www.npmjs.com/package/ngx-sse-client).

## Run sample

### Client

```
cd lib
npm run watch
```

```
cd lib
npm run start
```

### Server

```
cd server
npm run start
```
